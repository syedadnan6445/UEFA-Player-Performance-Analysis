⚽ UEFA Champions League Player Data Scraper

Project Overview

This project focuses on web scraping player statistics from the UEFA Champions League official website using automation tools.

The objective is to extract structured player-level data from a dynamic webpage and store it in a usable format for further analysis.

Tech Stack

Python
Selenium (Browser Automation & Web Scraping)
Pandas (Data Structuring & Export)

Data Source

UEFA Official Website:
https://www.uefa.com/uefachampionsleague/statistics/players/

Scraping Workflow

Launched automated browser using Selenium
Handled dynamic content (AG Grid table rendering)
Implemented scrolling to load all player records
Extracted data using CSS selectors
Cleaned and structured the extracted data
Exported dataset into CSV format

Extracted Data Fields
Rank
Player Name
Club
Position
Minutes Played
Matches Played
Goals
Assists
Distance Covered (km)
Top Speed (km/h)
